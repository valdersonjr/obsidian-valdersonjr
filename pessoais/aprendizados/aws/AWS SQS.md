# AWS SQS

SQS é uma fila gerenciada para **mensageria assíncrona** (desacoplar produtores e consumidores).

## Quando usar

- Buffer entre serviços
- Processamento assíncrono com retries

## Relacionado

- [[AWS Lambda]] (consumer comum)
- [[AWS SNS]] (fanout para filas)
- [[AWS EventBridge]] (eventos)
- [[AWS CloudWatch]] (métricas/alarms)
