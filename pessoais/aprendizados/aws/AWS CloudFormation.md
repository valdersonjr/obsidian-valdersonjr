# AWS CloudFormation

CloudFormation é **infraestrutura como código** (IaC) declarativa: você descreve recursos e a AWS cria/atualiza como um “stack”.

## Quando usar

- Padronizar criação de infraestrutura
- Repetibilidade e versionamento de ambientes

## Relacionado

- [[AWS CDK]]
- [[AWS IAM]] (permissões do deploy)
- [[AWS Systems Manager (SSM)]] (parametrização, em alguns cenários)
