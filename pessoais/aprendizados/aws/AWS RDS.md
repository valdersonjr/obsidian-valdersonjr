# AWS RDS

RDS é um serviço gerenciado de **banco relacional** (ex.: PostgreSQL, MySQL) com operações comuns abstraídas (backups, patching, etc.).

## Quando usar

- Você quer banco relacional sem operar tudo “na mão”.

## Conceitos

- **Instância** e **storage**
- **Multi-AZ** (alta disponibilidade)
- **Backups** e snapshots
