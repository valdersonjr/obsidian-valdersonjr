# AWS EBS

EBS é storage de **bloco** (volumes) geralmente acoplado a instâncias **[[AWS EC2]]**.

## Quando usar

- Disco para VM (root/data volume).
- Workloads que precisam de bloco (filesystem, DB em EC2, etc.).
