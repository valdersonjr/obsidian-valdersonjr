# AWS Organizations

Serviço para gerenciar **múltiplas contas AWS** como uma organização (governança, billing consolidado e políticas).

## Conceitos

- **Accounts** dentro da org.
- **OUs (Organizational Units)** para agrupar contas.
- **SCPs (Service Control Policies)** para impor limites máximos (guardrails).
