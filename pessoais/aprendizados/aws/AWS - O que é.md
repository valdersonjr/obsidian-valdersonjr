# AWS - O que é

AWS (Amazon Web Services) é uma plataforma de **computação em nuvem** que fornece serviços (compute, storage, banco, rede, segurança, etc.) sob demanda.

## Ideia central

Você “aluga” capacidade e serviços via API/console, paga pelo uso e consegue escalar sem operar datacenter próprio.
