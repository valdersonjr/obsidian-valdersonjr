# AWS Secrets Manager

Armazena e entrega **segredos** (ex.: senhas, tokens, strings de conexão) com rotação opcional e controle de acesso.

## Quando usar

- Você precisa de segredos com **controle de acesso** e **auditabilidade**.
- Quer rotação automática (com Lambda).
