# AWS DynamoDB

DynamoDB é um banco **NoSQL key-value/document** gerenciado, com escala automática e baixa latência.

## Quando usar

- Acesso por chave/partição, alta escala, latência baixa.
- Modelagem orientada a padrões de acesso (access patterns).

## Conceitos

- **Partition key** e **sort key**
- **Indexes** (GSI/LSI)

## Relacionado

- [[AWS Lambda]] (padrão comum: evento + persistência)
- [[AWS IAM]] (permissões)
- [[AWS CloudWatch]] (métricas)
- [[AWS KMS]] (criptografia, opcional)
