# AWS CloudTrail

CloudTrail registra chamadas de **API** e eventos de conta (quem fez o quê, quando, de onde).

## Quando usar

- Auditoria e investigação (security/compliance).

## Relacionado

- [[AWS IAM]]
- [[AWS Organizations]] (centralizar trilhas)
- [[AWS S3]] (destino comum para logs)
- [[AWS CloudWatch]] (observabilidade é diferente)
