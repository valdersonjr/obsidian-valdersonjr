# AWS Lambda

Lambda é computação **serverless**: você fornece código (função) e a AWS executa sob demanda, com escala automática.

## Quando usar

- Processamento por evento (fila, stream, HTTP).
- Jobs curtos e reativos (sem gerenciar servidor).

## Conceitos

- **Trigger** (evento), **timeout**, **memory/CPU**
- **Role de execução** (IAM)
