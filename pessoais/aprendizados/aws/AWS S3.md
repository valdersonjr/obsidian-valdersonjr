# AWS S3

S3 é storage de **objetos** (buckets e objetos) para arquivos, dados, backups e data lakes.

## Quando usar

- Arquivos estáticos, uploads/downloads, logs, backups.
- Integrações com analytics e eventos.

## Conceitos

- **Bucket**, **object key**
- **Policies** (bucket policy), **ACLs** (menos comum)
- Criptografia com **[[AWS KMS]]** (opcional)

## Relacionado

- [[AWS IAM]]
- [[AWS KMS]]
- [[AWS CloudFront]] (CDN na frente do S3)
- [[AWS Lambda]] (eventos do S3)
- [[AWS CloudTrail]] (auditoria de API)
