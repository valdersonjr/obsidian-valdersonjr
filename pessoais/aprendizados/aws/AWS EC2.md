# AWS EC2

EC2 é computação baseada em **máquinas virtuais** (instâncias) para rodar sistemas e aplicações.

## Quando usar

- Você precisa de controle do SO/runtime.
- Workloads stateful/legados, ou quando containers/serverless não encaixam.

## Conceitos

- **AMI**, **Instance type**, **Auto Scaling**
- **Security Group** (entrada/saída)
- Storage comum: **[[AWS EBS]]**
