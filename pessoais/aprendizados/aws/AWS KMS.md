# AWS KMS

KMS (Key Management Service) gerencia **chaves criptográficas** para criptografia (em repouso e, em alguns casos, operações de chave).

## Para quê serve

- Criar/gerenciar **CMKs** (chaves gerenciadas).
- Controlar uso via **policies** e **IAM**.
- Integrar com serviços (ex.: S3, EBS, RDS) para criptografia.
