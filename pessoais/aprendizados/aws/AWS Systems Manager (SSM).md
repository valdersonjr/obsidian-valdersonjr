# AWS Systems Manager (SSM)

SSM é um conjunto de ferramentas para **operar e automatizar** servidores e recursos (inventário, execução remota, patching, automações).

## Quando usar

- Acessar/operar instâncias sem SSH direto (Session Manager, quando habilitado).
- Automatizar tarefas operacionais.

## Relacionado

- [[AWS EC2]]
- [[AWS IAM]] (permissões/roles)
- [[AWS CloudWatch]] (logs/métricas, dependendo da integração)
