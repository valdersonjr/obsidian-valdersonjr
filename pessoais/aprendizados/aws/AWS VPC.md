# AWS VPC

VPC (Virtual Private Cloud) é a **rede lógica isolada** onde seus recursos rodam (subnets, rotas, gateways, segurança de rede).

## Conceitos

- **Subnets** (por AZ)
- **Route tables**
- **Internet Gateway / NAT Gateway**
- **Security Groups** (stateful) e **NACLs** (stateless)
