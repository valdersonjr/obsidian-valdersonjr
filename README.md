# Caderno de anotações

## Estrutura

- `trabalho/`: notas de trabalho
- `pessoais/`: notas pessoais

## Como usar (bem simples)

- Crie arquivos `.md` onde fizer sentido (`trabalho/` ou `pessoais/`).
- Use `[[links]]` entre notas quando houver relação.
- Se quiser seguir um método: veja [[Metodologia - Zettelkasten]].
